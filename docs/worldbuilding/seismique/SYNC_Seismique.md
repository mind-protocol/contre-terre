## RISQUES IDENTIFIES

| Risque | Severite | Mitigation |
|--------|----------|------------|
| Repetitivite du tremens de Nandi | Haute | Varier les manifestations : nausee → vomissement → tremblements → hallucinations → fusion avec le monde. Chaque chapitre doit avoir une "texture" de tremens differente. |
| Oubli du fond sismique dans les scenes emotionnelles | Moyenne | Checklist VALIDATION V1/V3 a appliquer a chaque scene de plus de 2 paragraphes. |
| Inflation spectaculaire des seismes | Moyenne | Les seismes ne doivent pas devenir des scenes d'action. L'important n'est pas le spectacle mais l'effet sur les corps et le Contact. |
| Incherence du tremens entre chapitres | Haute | VALIDATION V5 : le tremens ne regresse jamais durablement. Relire le chapitre precedent avant d'ecrire le suivant. |
| Le sismographe de Sihle oublie | Basse | Mentionner le sismographe dans chaque scene sismique significative (Ch. I-V). |
