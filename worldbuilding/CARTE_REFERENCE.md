## Disposition spatiale

Le monde est centré sur **le Volcan** (La Bouche / Le Creux / L'Embouchure). Sept archipels de roche stable entourent le volcan, séparés par un **désert sismique** (magnitude 4-5 de fond). Les caravanes suivent des **veines de roche stable** entre les archipels.

## Les 7 Archipels — Résumé pour le seeding

### 1. SUD — Les Basses-Terres

| Propriété | Valeur |
|-----------|--------|
| Roche | Basalte dense (crête profonde 300m) |
| Magnitude | 3.5 (amortissement naturel) |
| Contact | Lent, profond, étiré. Dialecte le plus ancien. |
| Architecture | Dômes aplatis cuir/basalte, réservoirs souterrains |
| Population | ~4 250 |
| Spécialités | Commerce, cordage, eau, Consortium |

**Lieux :**
- **Kahlamba** (~4 000) — Ville principale. Marché le plus grand. Siège du Consortium. Départ des expéditions. [Origine: Senzo]
- **Ndaba** (~200) — Tisserands de cordage. Meilleures cordes du monde. Fournisseur d'Inyoni.
- **Lesedi** (~50) — Pêcheurs d'eau. Descendent dans les fissures phréatiques. [Jabu y est passé]

### 2. NORD-EST — Les Sèches

| Propriété | Valeur |
|-----------|--------|
| Roche | Schiste feuilleté (amplifie les ondes) |
| Magnitude | 4.5 (amplifié) |
| Contact | Rapide, bref, nerveux. Rafales de 3-5 pressions en 2 secondes. |
| Architecture | Tours hautes, planchers suspendus, passerelles de corde |
| Population | ~2 140 |
| Spécialités | Écouteurs, prédicteurs, tissus d'aéromaître |

**Lieux :**
- **Mhluzi** (~2 000) — Sur la Table (plateau schiste). Réseau aérien de passerelles. Formation écouteurs/prédicteurs.
- **Iziphuku** (~60) — Village-tube sur lame de schiste. 300m long, 8m large. Contact permanent. [Origine: Nandi]
- **Phakade** (~80) — Tissus pour aéromaîtres. Fibres de plantes vibrantes. [Tissu de Thabo]

### 3. OUEST — Les Roulantes

| Propriété | Valeur |
|-----------|--------|
| Roche | Grès migrant (blocs qui dérivent de cm/jour) |
| Magnitude | Variable (mouvement tectonique oblique) |
| Contact | Contextuel, dépend de la direction de dérive. Le plus sophistiqué. |
| Architecture | Ponts de corde ajustés quotidiennement |
| Population | ~1 620 |
| Spécialités | Cartographie corporelle, survie, adaptation |

**Lieux :**
- **Ingwe** (~1 500) — 3 blocs de grès dérivant ensemble. Ponts de corde entre blocs. [Senzo formé par un cartographe d'ici]
- **Zama** (~120) — Bloc solitaire. Survivalistes. [Enama: 2 saisons]

### 4. NORD — Les Froides

| Propriété | Valeur |
|-----------|--------|
| Roche | Granite dense, froid, silencieux |
| Magnitude | 4 (basses profondes, spectre étroit) |
| Contact | Lourd, appuyé, presque martial. Pression intense. |
| Architecture | Cubes arrondis de granite massif (0.5m épaisseur), 5 niveaux |
| Population | ~3 080 |
| Spécialités | Taille de roche, mineurs, instruments de mesure |

**Lieux :**
- **Umzimkhulu** (~3 000) — Plus ancienne colonie. 5 niveaux (les plus vieux en bas). Essieux de chariots. Sismographes. [Origine: Sihle]
- **Mantolo** (~80) — Tailleurs de roche. Utilisent les séismes pour fendre le granite. [Sihle: mécanique de la roche]

### 5. CENTRE-SUD — Les Chaudes

| Propriété | Valeur |
|-----------|--------|
| Roche | Basalte jeune, coupant, noir |
| Magnitude | 4.5-5.5 (bas de l'Éveil) |
| Contact | Urgent, réactif, geste de vérification constant (*tu es là ?*) |
| Architecture | Camp semi-mobile. Dômes sur chariots, démontage en 6h. |
| Population | ~840 |
| Spécialités | Géologie, aéromaîtres, avant-poste Consortium |

**Lieux :**
- **Bhekisisa** (~800) — Camp mobile. Avance/recule au rythme du volcan. Dernier relevé sismique avant descente. [Origine: Enama, Thabo]
- **Themba** (~40) — Gardiens de l'entrée souterraine. Cartographes de l'entre-deux. Entretiennent les filaments bioluminescents.

### 6. EST — Les Salines

| Propriété | Valeur |
|-----------|--------|
| Roche | Gypse cristallin (friable, se dissout/recompose) |
| Magnitude | 4-5 standard |
| Contact | Cristallin, pointu. Bout des phalanges, pas la paume. Abrasif. |
| Architecture | Dépression de cristaux réfléchissants. Voiles de protection. |
| Population | ~1 260 |
| Spécialités | Sel, explosifs, dextérité, lumière |

**Lieux :**
- **Umsizi** (~1 200) — Ville la plus lumineuse. Cristaux de gypse réfléchissant. Marchands de voiles.
- **Nkosazana** (~60) — Extraction du sel. Gants de cuir (seules personnes à protéger leurs mains). [Inyoni: 1 an d'apprentissage]

### 7. SUD-OUEST — Les Creuses

| Propriété | Valeur |
|-----------|--------|
| Roche | Calcaire karstique (poreuse, cavités naturelles) |
| Magnitude | 4-5 standard |
| Contact | Contact hydraulique (variations de débit = messages). Vertical. |
| Architecture | Iceberg: 1/3 surface, 2/3 souterrain. Enfants grimpent avant de marcher. |
| Population | ~950 |
| Spécialités | Spéléologie, cartographie souterraine, eau |

**Lieux :**
- **Sibaya** (~900) — 37 grottes connectées sous une plateforme calcaire. Va-et-vient surface/souterrain. [Origine: Jabu]
- **Mathamo** (~50) — Cartographes tactiles. Cartographie du corps (topologie par Contact sur le bras). [Senzo: formé ici]

## Zones notables (hors archipels)

| Zone | Position | Nature | Importance |
|------|----------|--------|------------|
| **Désert sismique** | Entre tous les archipels | Dalles de basalte, sable, failles permanentes. Bruit blanc tellurique. | Routes caravanières = veines de roche stable |
| **Dents du Désert** | Désert (visible de 3 archipels) | Colonnes de basalte 5-20m, subduction ancienne. Dent Noire fendue. | Repères caravaniers |
| **Faille de Khensi** | Entre Sud et Centre-Sud | Déchirure 30m × 200m, fond invisible. Infra-basses 0.3 Hz. | Lieu de formation des écouteurs. [Sihle: 6 mois] |
| **Champs de Soufre** | Entre Chaudes et piémont | Sol jaune, croûteux, H2S/SO2. | Examen final des aéromaîtres. [Thabo: dernier mois] |
| **Piémont volcanique** | Flancs du volcan | Gravier basalte tranchant, ravines, +5-10°C, soufre. Fin de la lumière naturelle. | Transition surface → souterrain |
| **Entrée du réseau** | Piémont | Faille 3m × 1m, 30° de pente. Premiers filaments bioluminescents. | Début du monde souterrain |
| **Village des Sourds** | Souterrain (sous le volcan) | 200 hab. sourds. Amortissement -40%. Contact 3 doigts. Bioluminescence domestiquée. Cycle 6h. Contact hydraulique. | Le geste inconnu. Dernier lieu humain. |

## Paramètres clés pour le seeding

### Par lieu : propriétés à inclure dans chaque noeud `space`

```yaml
id: space_{subtype}_{nom}
type: archipel | ville | village | zone | biome | corridor
content: description narrative (prose)
synthesis: résumé 1-2 phrases embeddable
properties:
  roche: type de roche dominante
  magnitude_fond: float (3.5 - 5.5)
  spectre: basses | larges | hautes | mixte
  temperature_sol: °C estimé
  population: int (si habité)
  dialecte_contact: description courte
  luminosite: soleil | bâtons_chimiques | bioluminescence | noir
  eau: abondante | rare | souterraine | aucune
  specialisation: [liste des métiers dominants]
  archipel_parent: id de l'archipel
```

### Par citoyen : propriétés minimales

```yaml
id: actor_citoyen_{nom}
type: citoyen
content: description narrative
synthesis: résumé embeddable
properties:
  age: int
  archipel_origine: id
  lieu_residence: id
  metiers: [liste]
  guildes: [liste]
  dialecte_contact: description
  tremens_signature: description
  calibration: basses | hautes | larges
```
