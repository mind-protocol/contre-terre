# Siphelele

Le sel sait. Mes mains le dosent a l'instinct -- le sel de Nkosazana dans la bouillie de Kahlamba, la quantite exacte, jamais trop, jamais trop peu. Ca inquiete les superstitieux. Trop saler est un signe que la terre monte, disent les vieux. Moi je ne surdose jamais. Alors on me regarde avec des yeux de gens qui ne savent pas si c'est rassurant ou terrible.

Vingt-neuf ans, cuisinier du marche, Guilde du Lien. Je gonfle les joues quand je goute -- les enfants m'imitent dans tout le marche, une rangee de petites joues gonflees qui jaugent la bouillie comme si c'etait un art. C'en est un. Ma bouillie est la reference : epaisse, grasse, salee juste, avec l'arriere-gout de tubercule fermente que les gens d'ici trouvent reconfortant. Mzamo repare les hamacs a cote de mon echoppe. Mon pas est rapide pour un homme du Sud. Le marche est mon sol.
