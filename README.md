# Contre-Terre

> La horde marche contre le vent ; l'expédition descend contre la terre.

Roman en cours d'écriture. Miroir de *La Horde du Contrevent* (Alain Damasio).

## Prémisse

Un monde de séismes permanents. Le sol ne s'arrête jamais de trembler. Les humains ont développé un langage tactile — **le Contact** — parce que le son est devenu inutile. 7 personnes descendent dans un volcan pour pré-déclencher un séisme de magnitude 11. Ils meurent un par un. Le dernier active la charge.
